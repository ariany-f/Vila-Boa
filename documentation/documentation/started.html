WowDash

WowDash - Laravel - Admin Dashboard Template
How to Install Admin Templates
Install Laravel 11

If you're using XAMPP, WAMP, or MAMP, Laravel 11 should already be included. Make sure your server environment (Apache, Nginx, etc.) is configured to work with Laravel 11. If Laravel 11 is not installed, you can download it from the official Laravel 11 website.

2.Download and Extract the Zip File:

Download the zip file containing all files integrated with WowDash. Extract the contents of the zip file to your desired location.

3.Install Node JS

Make sure to have Node.js installed and running on your computer. If you don't have Node.js installed, you can download and install it from official website.

4.Install Node Modules

Once you are in your project directory, run the following command to install the necessary node modules, including those required for SASS compilation

npm install
5.Generate CSS using SASS compiler

To generate CSS from SCSS, execute the command

npm run dev

If you want to run compiler in watch mode run

npm run build

6.The development server is accessible at http://localhost:8000. To run on other port just run command : php artisan serve --port=8001

php artisan serve
